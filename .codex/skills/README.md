Tato složka obsahuje repo-scoped workflow skills pro Codex. Každý `aiscr-<name>` adresář je tenký vstup do kanonického workflow zdroje a jeho plánu.

## Co zde platí

- Skills doplňují `AGENTS.md`, `CODEX.md`, `.codex/README.md` a `.codex/GOVERNANCE.md`.
- Slug `aiscr-<name>` musí zůstat stejný napříč všemi zrcadlenými vendor stromy.
- Detail workflow patří do kanonického dokumentu a plánu, ne do lokálního README.
