VYPIS config
============

Modul config.

Funkce
------

.. py:function:: get_config(name)

   Vrací config. v aplikaci.

   **Parametry:**

   - ``name``: Parametr ``name`` předává se do volání ``get()``, vstupuje do návratové hodnoty.

   **Návratová hodnota:**

   Vrací výsledek volání ``get()``.
