**Custom repo subagents:** Markdown with YAML frontmatter under **`.gemini/agents/*.md`** ([Creating custom subagents](https://geminicli.com/docs/core/subagents/)). **Built-in** CLI subagents (`codebase_investigator`, `cli_help`, …) and global overrides: **`agents.overrides`** in **`settings.json`**.

**AIS CR hub:** **`aiscr-*`** files are generated by **`python .agents/scripts/generate_agent_definitions.py`** from the same SSOT as **`.cursor/agents/`**, **`.claude/agents/`**, and **`.codex/agents/`** (you may add more agents manually). Sync baseline: **`repos.toml`** / **`sync_policy.py`**.

Official docs: [Subagents](https://geminicli.com/docs/core/subagents/). Path map: **`agent_tool_feature_matrix.md`**, **`mandatory_vendor_doc_urls.toml`**.
